### Exercise 8: 事件循环与任务调度

理解了事件循环的核心概念：
- 事件循环是单线程的（和 JS 一样）
- `await` 会"让出控制权"，允许其他任务执行
- `asyncio.gather()` 让多个任务"交替"执行

完成了任务调度器练习：
```python
async def task_schedule():
    async def demo_task(seconds: float):
        await asyncio.sleep(seconds)
        return f"Task {seconds} done"

    task1 = asyncio.create_task(demo_task(1))
    task2 = asyncio.create_task(demo_task(2))
    task3 = asyncio.create_task(demo_task(3))

    print("等待中...")
    return await asyncio.gather(task1, task2, task3)
```

超时处理：使用 `asyncio.wait_for(coro, timeout)` 实现，类似 JS 的 `Promise.race()`。

### Exercise 9: 异步上下文管理器

理解了 `async with` 的工作原理：
- 自动调用 `__aenter__` 和 `__aexit__`
- JS 没有原生支持，需要用 try/finally 模拟

完成了异步文件处理器练习：
```python
class AsyncFileHandler:
    def __init__(self, filename):
        self.filename = filename

    async def __aenter__(self):
        await asyncio.sleep(0.1)
        print(f"打开文件: {self.filename}")
        return self

    async def __aexit__(self, exc_type, exc, tb):
        print(f"关闭文件: {self.filename}")
        return False

    async def read(self):
        await asyncio.sleep(0.2)
        return f"content of {self.filename}"
```

### Exercise 10: Python async vs JS async 完整对比

完成了带重试的异步请求练习：

```python
async def fetch_with_retry(url: str, max_retries: int = 3) -> Optional[dict]:
    for i in range(max_retries):
        try:
            res = await random_fetch(url)
            return {"result": res}
        except ValueError as e:
            await asyncio.sleep(1)
            print(f"捕获错误: {e}")
```

## 💡 关键收获

1. **执行方式差异**：JS async 函数调用后立即执行；Python 需要用 `asyncio.run()` 或 `await` 才会执行
2. **时间单位差异**：JS setTimeout 用毫秒，Python asyncio.sleep 用秒
3. **任务创建**：Python 需要显式 `asyncio.create_task(coro)`，JS 直接调用就自动开始
4. **上下文管理**：Python 有 `async with` 原生支持，JS 没有对应概念
5. **并发模式一致**：`asyncio.gather()` 和 `Promise.all()` 概念相同，都是并发执行取最长耗时

## 📊 Python async vs JS 对比速查表

| 场景 | JavaScript | Python |
|------|------------|--------|
| 定义异步函数 | `async function fn() {}` | `async def fn():` |
| 调用异步函数 | `fn()` 自动执行 | `fn()` 返回协程对象 |
| 等待结果 | `await promise` | `await coroutine` |
| 运行入口 | 自动运行 | `asyncio.run(main())` |
| 并发执行 | `Promise.all([...])` | `asyncio.gather(...)` |
| 创建任务 | 直接调用 | `asyncio.create_task(coro)` |
| 延迟 | `setTimeout(fn, ms)` 毫秒 | `asyncio.sleep(seconds)` 秒 |
| 超时 | `Promise.race([...])` | `asyncio.wait_for(coro, timeout)` |
| 错误处理 | `try/catch` | `try/except` |
| 上下文管理 | 无原生支持 | `async with` |
