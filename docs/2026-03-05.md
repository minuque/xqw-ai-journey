### 3. 关键字深入学习

**`pass`：**
- 空操作语句，函数未实现时占位
- 实现后可删除
- 注意：pass 之后的代码不会执行

**`assert`：**
- 断言语句，测试函数行为
- 条件为 False 时抛出 AssertionError
- 建议保留用于回归测试

**`@dataclass`：**
- 自动生成 `__init__`, `__repr__`, `__eq__`
- 适合简单的数据对象（Model、DTO）
- 不加的话需要手动写构造函数

### 4. 概念深入理解

**元组 vs 可迭代对象：**
- 可迭代对象：能用 for 循环遍历（list, str, tuple, set, dict, range）
- 元组：不可变的序列，可作为 dict 的 key
- 列表：可变的序列，不可作为 dict 的 key

**元组作为 dict key：**
- `args` 是元组，可以直接作为 key
- 优化：用 `frozenset(kwargs.items())` 兼容 kwargs

**装饰器原理：**
- 本质是一个函数，接收函数作为参数，返回包装后的函数
- 带参数装饰器：装饰器工厂模式（返回装饰器的函数）
