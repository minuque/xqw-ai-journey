## Why

一名具有 5 年 Vue 前端开发经验和 Java 后端经验的工程师，计划转型 AI 应用开发（非算法方向）。需要一个结构化的学习路径来系统掌握 Python + FastAPI 后端开发、Claude API 对接、Prompt 工程、RAG 系统搭建、以及 LangChain/LangGraph Agent 工作流构建。通过项目驱动学习，每个阶段都有明确的产出物，确保学习效果可衡量。

## What Changes

- 建立 Python + FastAPI 后端开发能力，替代 Java 成为 AI 应用的主力后端语言
- 掌握 Claude API（Anthropic SDK）的完整使用，包括 Messages、Streaming、Tool Use、Vision
- 系统学习 Prompt 工程，包括 Few-shot、CoT、结构化输出、Claude 特有技巧（XML 标签、Prefill、Extended Thinking）
- 从零搭建完整 RAG 系统：文档解析 → 分块 → Embedding → 向量检索 → Reranking → LLM 生成
- 使用 LangChain/LangGraph 构建复杂 Agent 工作流，包括多 Agent 协作和 Human-in-the-loop
- 完成至少 1-2 个可落地的综合实战项目

## Capabilities

### New Capabilities

- `python-fastapi-foundation`: Python 语言快速上手 + FastAPI Web 框架开发能力，包括环境管理（uv）、Pydantic 数据模型、异步编程、SSE 流式输出
- `claude-api-integration`: Claude API 完整对接能力，包括 Messages API、Streaming、Tool Use、System Prompt 设计、Vision 多模态
- `prompt-engineering`: Prompt 工程体系化能力，包括核心技巧（Few-shot、CoT、结构化输出）、Claude 特有技巧、Prompt 管理与评估
- `rag-system`: 完整 RAG 系统搭建能力，包括文档处理、分块策略、Embedding（Voyage AI）、向量数据库（Chroma/Qdrant）、检索优化、Reranking、评估（RAGAS）
